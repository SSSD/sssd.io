Community
=========

.. poster-hints::
    :class: poster-community

    .. poster-hint::
        :title: GitHub
        :icon: ./_static/images/icon-github-white.svg

        The SSSD source code is hosted on `GitHub <https://github.com/SSSD>`_.

        Visit the `project page <https://github.com/SSSD/sssd>`_ to submit
        issues and pull requests.

    .. poster-hint::
        :title: Mailing Lists
        :icon: ./_static/images/icon-mailinglists.svg

        `sssd-users <https://lists.fedorahosted.org/archives/list/sssd-users@lists.fedorahosted.org>`_
        is for user questions and community support.

        `sssd-devel <https://lists.fedorahosted.org/archives/list/sssd-devel@lists.fedorahosted.org>`_
        is for developer and contributor coordination and discussion.

    .. poster-hint::
        :title: IRC
        :icon: ./_static/images/icon-irc.svg

        We use the **#sssd** channel on `libera.chat <https://libera.chat>`_.

        Feel free to ask both user and contributor questions.

.. rst-class:: community-bug

    Have you found a bug in SSSD? Please report it in our `issue tracker`_.

.. _issue tracker: https://github.com/SSSD/sssd/issues


.. rst-class:: community-text

    Whether you use SSSD, wish to contribute to improve it or wish to follow its
    development more closely, you are welcome to join.
